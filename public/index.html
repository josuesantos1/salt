About Salt

Salt is a modern, lightweight, and high-performance HTTP server built for developers who value speed, security, and simplicity.

From static sites to high-traffic APIs, Salt delivers production-grade features with a configuration you can understand in minutes.

Features

  • Blazing fast and secure by default
  • Minimal resource usage
  • Easy to configure and deploy
  • Production-grade for static sites and APIs
